# modules/pai — PAI-Additive Source

This folder is the source of truth for all PAI customizations this box adds on top of upstream [danielmiessler/PAI](https://github.com/danielmiessler/PAI): custom hooks, skills, agents, lib, Algorithm, CLAUDE.md, settings.

## Layout

- `CLAUDE.md`, `CLAUDE.md.template` — global Claude Code instructions
- `settings.json.template` — settings.json with secrets replaced by `${VAR}` placeholders
- `link.sh` — symlink the subtrees below into `~/.claude/` (idempotent)
- `install-settings.sh` — render `settings.json.template` → `~/.claude/settings.json` using BWS/env
- `install.sh` — upstream PAI v4.0 installer (kept as-is for parity with danielmiessler/PAI)
- `PAI/` — Algorithm, PRDFORMAT, CONTEXT_ROUTING, AISTEERINGRULES
- `PAI-Install/` — install wizards / upgrade assets
- `skills/` — Skill definitions (loaded by Claude Code via `~/.claude/skills` symlink)
- `hooks/` — Lifecycle hooks (referenced by settings.json)
- `agents/`, `lib/`, `VoiceServer/`, `Observability/`, `WORK/`

## Install on a fresh box

```bash
cd ~/repos/imladris
bash scripts/install-deps.sh   # fetches Ruflo (npm) + PAI upstream (git clone)
bash modules/pai/link.sh       # symlinks modules/pai/* -> ~/.claude/*
```

The `install.sh` script is idempotent — re-run any time.

## What stays in `~/.claude/` (NOT tracked by git)

Runtime state and personal content live only on the box and are excluded from git:

Personal content (never committed — back up separately, encrypted):
- `USER/` — user bio, resume, contacts, preferences
- `TELOS/` — life goals, beliefs, mission

Runtime state (regenerated by hooks and session activity):
- `MEMORY/` — PRDs, learning reflections, work state
- `projects/` — raw session transcripts (*.jsonl)
- `sessions/`, `shell-snapshots/`, `paste-cache/`, `History/`, `tasks/`
- `cache/`, `logs/`, `telemetry/`, `state/`, `debug/`, `file-history/`
- `todos/`, `backups/`, `plugins/`, `ai-failover-state/`, `session-env/`
- `history.jsonl`, `settings.local.json`, `settings.json` (rendered from template)
- `ai-provider.env`, `.credentials.json`, `.update.lock`, `mcp-needs-auth-cache.json`

Back these up separately (Postgres dump + S3) — never commit them.
